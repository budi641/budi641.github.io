The project incorporated detailed system design using sequence diagrams, ERDs, and class diagrams. Comprehensive documentation was produced, including SRS and SDS, and testing methods like unit testing, code smells analysis, and usability testing were employed. The platform was deployed on Microsoft Azure, utilizing App Services and Azure SQL, with automated deployments through CI/CD pipelines.

TASM-Space-Invaders

This project was developed to gain a deeper understanding of how low-level machine code translates into instructions and actions on the screen. It uses the 8086 architecture to create a version of the classic retro game "Space Invaders," written entirely in assembly language. The goal was to explore the fundamental workings of the CPU and its low-level architecture by directly interfacing with hardware and managing instructions at the machine level.
